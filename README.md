# Insurance-premium-predictor

## Project Overview

This project predicts health insurance premium amounts using data science techniques & Machine Learning algorithms. The dataset contains customer information such as age, gender, height, weight, and insurance coverage. Height and weight are used to calculate BMI (Body Mass Index), which is then used as an important feature for premium prediction.

## Results

Multiple machine learning models were evaluated for health insurance premium prediction. Among them, the Decision Tree Regressor performed best, achieving a training score of 100% and a testing score of 99%. Based on these results, the Decision Tree model was selected for premium prediction.
